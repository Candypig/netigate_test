# netigate_test

## 特徵工程:
我使用了兩種不同特徵(可以選擇所要使用的特徵):
1. 使用時間做為特徵:\
我將日期擷取出月份、日期(、小時，如果預測目標以小時為單位)，以此預測特定時間的用電量。
2. 時間序列:\
  我將前n筆(By天或小時)的用電量作為預測下一日(小時)的特徵。若要預測三天，則會將第一、二日的預測結果也加入特徵用以預測。

## 使用方法
將訓練資料(.csv)放置於row_data資料夾中。\
如要使用time series 方式預測，將所要預測日期前的資料放置於predict_needed_data資料夾中。\
執行main會以預設方式進行訓練與預測(需確保上述兩個資料夾具備資料)。\
可以依照不同預測目標、所需模型等進行調整。
